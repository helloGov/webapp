# helloGov

## Config
- ask wcrest for secrets.js file, and put into conf folder

## Install dependencies
- install node
  ```
  brew install node
  ```
- install mongodb
  ```
  brew install mongodb
  ```
- install packages
  ```
  npm install
  ```
- install sass
  ```
  sudo gem install sass
  ```

## Start app
- `npm run start-dev`
