# helloGov

## Config
- add `conf/secrets.js` file
  ```
  $ touch conf/secrets.js
  ```
- ask in slack channel for `secrets.js` file

## Install dependencies
- install node
  ```
  brew install node
  ```
- install mongodb
  ```
  brew install mongodb
  ```
- install packages
  ```
  npm install
  ```
- install sass
  ```
  sudo gem install sass
  ```

## Start app
- `npm run start-dev`
